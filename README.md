# TaskAPI

A simple Task Management API built with Java and Spring Boot.  
This project allows users to create, update, delete, and manage tasks with features like due dates, priorities, and categories.

## Features
- Create, update, delete tasks
- Mark tasks as completed
- Assign due dates and priority levels
- Categorize tasks (Work, Personal, Study)
- Multi-user support

## Tech Stack
- Java (Core & Advanced)
- Spring Boot (REST APIs)
- Hibernate ORM
- MySQL Database
